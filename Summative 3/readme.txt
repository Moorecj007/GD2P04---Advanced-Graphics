Readme

About:
This application creates an environment in DX10 where two graphics
techniques are demonstrated. The chosen techniques are animating 
transparent water and reflections.

Controls:
WASD 		-> Move the Camera along first person axes
Arrow Keys 	-> Rotate the Camera
E		-> Fly the Camera directly up
Q		-> Fly the Camera directly down
Esc		-> Quit the application
